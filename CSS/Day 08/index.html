CSS Position Demo

Static

Default position (normal document flow, no special behavior).

Relative

Moves relative to its normal position using top, left, etc.

Absolute

Positioned relative to the nearest ancestor with position: relative.

Sticky

Acts like relative until scrolled, then it “sticks” to the defined edge (e.g. top).

Scroll down to see the sticky element in action and the fixed element stay in place.

Fixed

Always stays in the same place relative to the viewport, even when scrolling.
